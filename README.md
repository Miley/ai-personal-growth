# 另一个我

一个给自己使用的成长 PWA：把私密陪伴、表达训练和英语阅读放在同一个轻量工具里。

在线使用：[https://miley.github.io/ai-personal-growth/](https://miley.github.io/ai-personal-growth/)

## 能做什么

- **另一个我**：用文字、图片或语音记录经历和困惑，获得贴近个人语境的回应与启发。
- **表达训练**：优先练即兴聊天与社交，再练向上沟通/说服、工作汇报/会议表达，最后才是正式演讲。
- **英语阅读**：按从易到难的材料进行阅读练习，面向旅行交流、日常英文材料和陪孩子的英语输入。
- **记忆带走**：在“记忆”页导出已确认保存的文字，通过隔空投送或“存储到文件”带到电脑。

## iPhone 使用

1. 用 Safari 打开上面的链接。
2. 点击“分享” → “添加到主屏幕”。
3. 从主屏幕打开“Another-Me”；首次使用图片或语音时允许对应权限。
